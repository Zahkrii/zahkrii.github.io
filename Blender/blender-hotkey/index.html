Blender 快捷键

2022-03-27
Blender 快捷键
Num 1 主视图
Tab 切换编辑模式
Ctrl + A 应用
滚轮 缩放
按住滚轮 自由旋转
Shift + 滚轮 平移
Ctrl + B 倒角
Alt + Z 透显模式
Shift + A 新建网格 / 添加 Shader 节点
Shift + S 改变原点中心
N 呼出侧边菜单
Num 0 相机视图
Ctrl + Alt + Num 0 活动相机对齐到当前视角
G 移动
S 缩放
R 旋转
Ctrl + Alt + Q 四视图
Num . or Home 适合视图大小
E 挤压出面
I 面内挤压
Shift + G 选择相似
Ctrl + Shift + B 点倒角
J 连接点
Z 着色方式
Shift + D 复制
Shift + 右键拖动 划分节点连线
