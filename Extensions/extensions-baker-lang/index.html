VS Code 扩展 - Baker Language Support

2022-06-06

一个简单的 VS Code 扩展插件，@Macaron-Lawrence 制作的 蜜饼（假）生成器 的配套工具。实现格式化代码，语法高亮，代码片段，自定义主题等功能。目前已上架 Visual Studio Marketplace.

下载 & 安装

Visual Studio Marketplace

直接前往 Visual Studio Marketplace 下载安装。

VS Code 扩展页面搜索安装

Ctrl + Shift + X 打开 VS Code 扩展页面，搜索：Baker Language Support 安装。

Github

前往 Github 下载 vsix 安装包，双击运行进行手动安装。

使用方法

文件名后缀：.cake

自定义主题

插件自带自定义主题 Baker，Ctrl + K, T 更改应用主题。

语法高亮

代码片段

  • 粗体 bb - <b></b>

  • 红色强调 rr - <red></red>

  • 空行 br - <br>

  • 叠甲声明 shield -（内容太长）

  • 图片 pic - &lcub;&lcub;&lcub;pic.png

  • 细体 ll - <l></l>

  • 白色字体 ww - <white></white>

右键菜单 & 快捷键

  • Alt + B 设为粗体（Bold）

  • Alt + R 红色强调（Red）

开源代码仓库

  • 假饼生成器/Fake Cake Baker：https://github.com/Macaron-Lawrence/fakeCakeBaker
